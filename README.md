# game-of-life
https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life

## install

```
brew install rye

rye sync
```

## run test

```
rye run pytest
```

## run

```
rye run game-of-life
```

### Game of Life controls
- Click on cells to toggle their state
- Space: Pause/resume simulation
- R: Randomize grid
- C: Clear grid
